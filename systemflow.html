System Flow

How will the user interact with the application by using the user interace?

Introduction

For our target group audience it will always be necessary to apply the Emotiv Headset at the start of the day. The dedicated screen that will run the Neural Home application and Emotiv BCI, this is mounted on the wall.

Scenario: User starts their day with turning on their lights.

Step 1- Caretaker applies the Emotiv Headset to the user's head.

Step 2- The caretaker clicks the Start button(Located on the dedicated screen) to connect to the websocket.

Step 3- The user raises their eyebrows to navigate to the Presets.

Step 4- The user clenches their teeth to select the Morning Preset.

Scenario: User wants to turn the living room lights red. (The Emotiv Headset is already applied to the user)

Step 1- The user uses the Mental Command to activate the navigation for the application. (The Neural Home application boots up and connects to the websocket.

Step 2- The user navigates to the Living room lights, by furrowing/raising eyebrows to go left and right, and clench their teeth to select the room.

Step 3- Once in the Living Room section, the user selects the “Red” Preset to turn the Living Room lights red.
